# About This Repository

This repository contains skills to use when authoring code in Discourse.

## Claude Code

You can register this repository as a Claude Code Plugin marketplace by running the following command in Claude Code:

```
/plugin marketplace add discourse/skills
```

Install plugin via:

```
/plugin install frontend-skills@discourse-agent-skills
```

After installing the plugin, you can use the skill by just mentioning it. For instance, if you install the `frontend-skills` plugin from the marketplace, you can ask Claude Code to do something like: "Use the frontend skill to refactor the CSS file /foo/bar.scss"
